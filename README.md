## Sample Projects

### [01 :  X Instruments MeasAPI v1](./01-x-instruments-measapi/)

API design review for a fictional LCR meter REST API. 8 issues identified and resolved:

- Critical: API key in URL query parameter, missing versioning
- High: inconsistent frequency units across endpoints, errors returned as HTTP 200, no pagination
- Medium: mixed naming conventions, implicit stateful device lifecycle
- Low: magic integers for measurement type

**Tools:** OpenAPI 3.0, YAML
